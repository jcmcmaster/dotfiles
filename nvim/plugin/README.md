## Order Matters

Files in this directory are sourced in alphabetical order, hence the numbering.

## Phases

### 0* - Before Plugins
Mostly configuration of nvim itself. Maps, opts, and autocmds.

### 1* - Plugins
Unless a plugin is cross-cutting such that it has to be loaded after most others, it belongs here.

### 2* - Higher Order Plugins
Plugins that wrap other plugins. Completion lives here.
